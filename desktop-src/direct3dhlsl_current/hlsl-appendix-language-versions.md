# Language Versions

## [`HLSL 2016`](hlsl-appendix-language-version_2016.md)

Adds support for [`int64_t and uint64_t`](hlsl-variables-dataTypes.md#scalar-types) and their [`vector`](hlsl-variables-dataTypes.md#vector-types) and [`matrix`](hlsl-variables-dataTypes.md#matrix-type) forms.

## [`HLSL 2017`](hlsl-appendix-language-version_2017.md)

Adds [`enum and enum class`](dx-graphics-hlsl.md) declarations.

## [`HLSL 2018`](hlsl-appendix-language-version_2018.md)

Adds width-specific types for [`floats`](hlsl-variables-dataTypes.md#float) and [`ints`](hlsl-variables-dataTypes.md#int), for 16, 32 and 64 bit widths.

## [`HLSL 2021`](hlsl-appendix-language-version_2021.md)

Adds support for various C++ style features.

- [`Template functions`](hlsl-language-functions.md#template-functions)
- [`Templated structs`](hlsl-language-type-struct.md)
- [`Bitfields on struct elements`](hlsl-language-type-struct.md#bitfields)
- [`operator overloading`](hlsl-language-statements.md#operator-overloading)
- [`C++ style function overloading`](hlsl-language-functions.md#c-style-function-overloading)
- [`Logical operation short-circuiting scalars.`](hlsl-language-statements.md#logical-operation-short-circuiting-for-scalars)
